#  Simple Rule-Based Chatbot 

##  Overview

This project is a beginner-friendly text-based chatbot built using Python. It responds to predefined user inputs such as greetings, questions, and farewell messages using simple conditional statements.

##  Features

* Interactive text-based conversation
* Predefined responses
* Continuous chat until the user types **bye**
* Easy to understand and modify
* Beginner-friendly project

##  Technologies Used

* Python 3

## Concepts Used

* Variables
* Strings
* `input()` and `print()`
* `while` loop
* `if`, `elif`, `else`
* String methods (`lower()`)

##  How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python chatbot.py
```

##  Example Conversation

```
===== Welcome to ChatBot =====

You: hello
Bot: Hi! Nice to meet you.

You: how are you
Bot: I am fine, thanks! How are you ?
You: I am fine 
Bot: that's great to hear 

You: what is your name
Bot: My name is Python Bot.

You: thank you
Bot: You're welcome!

You: bye
Bot: Goodbye! Have a nice day.
```
